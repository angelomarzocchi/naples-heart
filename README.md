

# Naples' Heart

Naples' Heart is an Android project to play with Android adaptive layouts.
It shows City recommentations divided into 3 categories: 
- Pizza
- Monuments
- Aperitif
  
The app uses the list-detail canonical layout to adapt to larger screens.
